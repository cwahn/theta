## Release v0.1.0-alpha.54

- **Fix: vite-plugin-theta no longer writes files when content is unchanged** — `type-gen.ts` and `umbrella.ts` previously used `fs.writeFileSync` unconditionally, touching `theta-actors.d.ts`, `Cargo.toml`, and `src/lib.rs` on every Vite build even when content was identical. This caused unnecessary rebuilds. A new `writeIfChanged` helper (in `fs-util.ts`) skips the write when the existing content matches, preserving mtimes.

- **Fix: Debug tracing for accepted bi-streams in `Peer`** — Added a `debug!(from = %self, "accepted bi-stream")` log line in `peer.rs` when a new bidirectional QUIC stream is accepted, making it easier to trace connection activity at the `debug` log level.
